MovieBox-style React + Vite + TypeScript scaffold (upgrade)

How to use:
1. unzip moviebox-upgrade.zip
2. npm install
3. npm run dev
4. npm run build (creates dist/)
5. Deploy dist/ to Netlify or Vercel

Notes:
- This is a frontend scaffold. Login is client-side demo only.
- Replace SAMPLE_CONTENT in src/services/data.ts with real API integration.
- Set VITE_API_BASE env var for API base URL if you integrate backend.
